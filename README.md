# practik
Программа представляет собой аналог функции `cp` в Linux. Реализует различные методы: `open()`, `read()`, `write()` и т.д.

##Структура проекта
- `code.c` - исходный код программы
- `makefile` - файл для сборки проекта
- `copy` - исполняемый файл
- `Hello.txt` - текстовый исходный файл
- `cp_Hello.txt` - тестовый целевой файл
## Сборка программы 
```bash
make clean
make
```
где `make clean` используется для очистки предыдущей сборки

## Использование
`./copy <исходный файл> <целевой файл>`
